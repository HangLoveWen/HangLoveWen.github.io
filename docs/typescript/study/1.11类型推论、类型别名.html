    类型推论

    马行 2/2/2023

    什么是类型推论 let str = "小满zs" 1.我声明了一个变量但是没有定义类型

    TypeScript 会在没有明确的指定类型的时候推测出一个类型，这就是类型推论

    所以TS帮我推断出来这是一个string类型

    不能够在赋值给别的类型

    2.如果你声明变量没有定义类型也没有赋值这时候TS会推断成any类型可以进行任何操作

    类型别名 type 关键字（可以给一个类型定义一个名字）多用于符合类型

    定义类型别名

    type str = string

let s:str = "我是小满"

console.log(s);
